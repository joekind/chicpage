## 🖋️ 愿景：简化文字表达

**ChicPage** 是一个简单的 Markdown 编辑器。在信息冗余的今天，我们希望通过**直观的排版控制**和**简洁的视觉设计**，帮助创作者更轻松地在微信和小红书等平台分享内容。

---

## ✨ 核心特性 (Key Features)

### 🎨 预设排版主题 (Layout Themes)
内置多套经过匹配的公众号排版方案，支持一键切换，让文章呈现更清晰。

### 📸 小红书切图工具 (Social Slicer)
提供便捷的切图功能，可以将 Markdown 文本转化为适合小红书展示的图片卡片。

### 🍱 网格化排版方案 (Grid Layout)
支持现代网格化布局，方便用户调整图片与文字的比例。

### ⚡ 简洁的创作空间 (Simple Workspace)
提供分屏预览功能，帮助用户在写作时实时查看排版结果。

---

## 🛠️ 技术栈 (Tech Stack)

ChicPage 采用了当前成熟的 Web 技术：

- **Next.js**：React 开发框架
- **Tailwind CSS**：样式管理
- **Framer Motion**：基础动画效果
- **Lucide Icons**：图标库
- **Zustand**：状态管理

## 🚀 快速开始 (Getting Started)

确保您的环境已安装 `Node.js 18+`。

1. **克隆仓库**
   ```bash
   git clone https://github.com/joekind/chicpage.git
   cd chicpage
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发环境**
   ```bash
   npm run dev
   ```
   访问 `http://localhost:3000` 即可开始。
